# Known Limitations

本文档集中记录 MistVault 初版已知限制。这里的限制是当前版本边界，不代表长期不会支持。

## OCR 限制

- 图片 OCR 仅支持 `jpg`、`jpeg`、`png`、`bmp`。
- OCR 对数学公式不稳定，不保证公式结构、上下标、分式、矩阵等准确。
- OCR 对手写字不稳定。
- OCR 对低清图片、倾斜图片、复杂背景图片可能不准确。
- 扫描版 PDF 第一版不支持自动 OCR。
- Word 文档中的图片第一版不做 OCR。
- OCR 结果应由用户检查和修正后再用于学习或 AI 讲解。

## 文档提取限制

- `txt` / `md` 支持文本提取。
- `docx` 只支持基础正文文本提取。
- `.doc` 不支持。
- `docx` 中复杂公式、图片、批注、脚注、页眉、页脚、复杂表格和复杂版式可能不完整。
- PDF 只支持文本层提取。
- PDF 图片内容不会被自动识别。
- 如果 PDF 是扫描版或图片型 PDF，提取结果可能为空。

## AI 限制

- AI 讲解依赖用户配置的第三方 provider 和 API Key。
- AI 调用需要联网，可能产生 provider 费用。
- AI 回答仅供学习参考，不保证完全正确。
- AI 默认只发送当前错题文本。
- 附件提取文本只有用户明确选择后才会发送。
- 不发送附件原文件、图片 base64、本地路径或整个错题库给 AI。
- Claude / Gemini 第一版不作为原生 adapter 支持。
- 当前不支持 AI 多模态附件输入。

## 打包与安装限制

- 初版本地测试安装包可能未签名。
- 未签名安装包可能触发 Windows SmartScreen 或未知发布者提示。
- `better-sqlite3` 是 native module，必须在打包后验证。
- OCR runtime 通过 `extraResources` 打包，发布前必须验证 `win-unpacked` 和 installer 中的 runtime 路径。

## 数据与数据库限制

- 用户数据默认保存在本地数据目录，不保存在安装目录。
- 卸载默认不删除用户错题库。
- MySQL 是高级预留配置，默认运行时仍是 SQLite。
- 当前不提供云同步、账号系统或跨设备自动合并。
